# python-data-visualization
A Python-based data engineering script that generates high-volume, randomized e-commerce transactional data, structures it into an optimized Pandas , and performs graphical distribution analysis using Seaborn.
# E-Commerce Data & Seaborn Plotting Practice

This is a personal learning project where I am practicing data generation with numpy , data manipulation with Pandas, and data visualization using Seaborn. 

## What This Script Does
1. Generates a random dataset between 100,000 and 999,999 rows of fake e-commerce shopping data (tracking items like phone, shaver, and protein powder).
2. Uses Pandas to structure the data into a clean table ( order_id , customer_id , item_price etc ) and calculate the total revenue.
3. Uses Seaborn to create a box plot chart that shows the price distributions across all categories.

## Why I Built This
I built this script to practice moving away from basic Matplotlib bar charts and learn how to use advanced Seaborn statistical plots instead. 

## Tech Stack Used
* Python 3
* Pandas
* NumPy
* Seaborn
* Matplotlib
